
#  Customer Billing System

A console-based **Customer Billing System** written in **C**, designed to help businesses manage customer records, generate invoices, and calculate totals with tax.

---
## How It Works

Customer Info — user enters name, address, and phone
Item Entry — for each item, user enters name, quantity, unit (optional), and price
Bill Generation — program calculates subtotal per item, applies 5% tax, and prints the final bill
