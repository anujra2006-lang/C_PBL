
#  Customer Billing System

A console-based **Customer Billing System** written in **C**, designed to help businesses manage customer records, generate invoices, and calculate totals with tax and discount support.

---

##  Features

- **Customer Management** — Add, edit, and delete customer profiles (name, address, contact details)
- **Bill Generation** — Create itemized bills for products or services
- **Total Calculation** — Automatically compute subtotals, taxes, and discounts
- **Invoice Records** — View and manage past billing entries
- **Data Persistence** — Store customer and billing data using file I/O

---

##  Project Structure

```
customer-billing-system/
├── main.c               # Entry point and main menu
├── customer.c           # Customer management functions
├── customer.h           # Customer struct and function declarations
├── billing.c            # Bill generation and invoice logic
├── billing.h            # Billing struct and function declarations
├── utils.c              # Helper/utility functions
├── utils.h              # Utility function declarations
├── data/
│   ├── customers.dat    # Stored customer records
│   └── bills.dat        # Stored billing records
├── Makefile             # Build configuration
└── README.md
```
